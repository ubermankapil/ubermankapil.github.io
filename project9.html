Summary of Angular Restaurant Menu


Angular Restaurant Menu


Angular JS web application to display the menu of a Restaurant. Used various Angular directives like ng-app, ng-repeat, ng-init, ng-model, ng-class, ng-show to construct the menu of a restaurant.
Also used Angular filters to filter out only the required result like filter results by rating, date or author. Got a lot of dynamic functionality in the page. Also created Angular form for submitting the feedback of the Restaurant and also included form Form Validation feature using Angular JS and not by HTML5.
