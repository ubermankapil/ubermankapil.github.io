Summary of Stock Market Analyzer


Stock Market Analyzer


The main objective of this project is to help Investors and Share-Brokers in taking the decision between any 2 Companies while Purchasing, Selling or Retaining shares through an Android Application.

This is a team project. We have a team of 4 members including me. This projet we have started from December 2015 and it is still going on, as it is little vast because of much work involved.

There will be 3 main activities for this Android application. One activity for the Purchasing shares section, Second activity for Selling shares section and the last activity for the Retaining shares section. In each of these 3 section, user will input the 2 major companies in which he/she is interested but confused that which one will be the better ? For this, We are using Google Finance API in my app, to get all live current data. This data will be in JSON format. We will extract the importnat values required from the JSON data which is send in response by calling the Google Finance API, by doing JSON Parsing. We are writing the code which will answer these 3 questions :
Q1) Why is it better to purchase shares of company_A rather than company_B?
Q2) Why is it better to sell shares of company_A rather than company_B?
Q3) Why is it better to retain shares of the company_A or company_B or for both?
Also, the code for these 3 questions will be constructed by considering as many possible factors we can take into consideration, because more the number of factors taken into consideration more will be the accuracy of the final result. Also, we will consider history and some remarkable trends observed in the past of each company into consideration while giving the final result.
